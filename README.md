# astrbot_plugin_force_silent

强制静默插件，支持硬静默与协同模式。

## 指令

- `/强制静默 状态`
- `/强制静默 统计`
- `/强制静默 开启`
- `/强制静默 关闭`
- `/强制静默 添加群 <群号>`
- `/强制静默 删除群 <群号>`
- `/强制静默 协同开启`
- `/强制静默 协同关闭`

英文兼容：`/force_silent ...`

## 模式说明

- 协同模式（默认）：不 `stop_event`，适合与采集插件并存
- 硬静默模式：命中群后直接 `stop_event`

建议：与 `chat_export` 同时使用时保持协同模式开启。
